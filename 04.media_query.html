響應式網頁設計(Responsive web design, RWD)

  • 1. 響應式網頁設計，讓內容如水一般能適應各種裝置的容器，能在每個裝置尺寸正常瀏覽
  • 2. 媒體查詢的斷點(break)為不同裝置尺寸設定通過的位置點，當通過設定的位置點套用該尺寸的樣式
  • 3. 響應式一般使用的單位為rem、%、vh、vw等相對單位，字體大小會使用rem，容器的width、height會以%為主
  • 4. 寫的時候行動裝置優先，從最小尺寸開始思考內容的設計
  • 5. 文件一定要有這行才能做RWD：<meta name="viewport" content="width=device-width, initial-scale=1.0">
